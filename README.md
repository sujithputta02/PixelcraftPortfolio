## ✳︎ Visual Experience & Showroom Core

PixelCraft is an ultra-luxury digital gallery designed to elevate graphic design and visual storytelling concepts into interactive, museum-grade spectacles. It operates under a dedicated **Discomorphism** design system—blending highly reflective specular glares, deep liquid-obsidian mirror panels, and smooth iridescent animations.

### 💎 Signature Features

1. **Cinematic Entrance Preloader**
   * A premium, high-contrast entrance preloader that counts from `000%` to `100%` and seamlessly fades and slides up to reveal the showroom with hardware-accelerated Bezier ease.
2. **Bangalore Live Status Ticker**
   * Located at the head of the showroom, displaying a real-time updating digital clock synced with Bangalore's timezone (`Asia/Kolkata` • `GMT+5:30`) and an active locator beacon.
3. **Iridescent Platinum Typography**
   * The primary `SUJITH` header signature is styled in a custom, slow-shimmering metallic linear gradient, reflecting specular twilight light.
4. **3D Mouse Parallax & Specular Spotlights**
   * Profile, About, and Testimonial cards are bound to smooth 3D mouse parallax hooks. Moving the cursor over the elements projects relative radial spotlight glares, mimicking shifting physical light on glass.
5. **Interactive Exhibit Carousels**
   * The high-fidelity exhibits support touch-swipe gestures on mobile devices and feature glassmorphic navigation buttons and segmented progress indicators on desktop.
6. **Dynamic Inquiries Channel**
   * A fully interactive contact dashboard equipped with service capsule selectors (with a custom "Other" specify text field), project timeline indicators, a real-time character brief counter, and automated `mailto:sujithputta02@gmail.com` client payload triggers.

## 🛠 Tech Stack & Systems

* **Core Engine**: React 18, TypeScript, Vite
* **Design Language**: Tailwind CSS & Vanilla CSS (Luxury Obsidian & Specular Glow parameters)
* **Animation & Rendering**: HTML5 Canvas Trailing Glows, CSS Specular Noise Grain Loops, and custom Bezier easing formulas.

## ⚡️ Quick Start

Initialize the development environment locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sujithputta/pixelcraft.git
   cd pixelcraft
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Launch the development server**:
   ```bash
   npm run dev
   ```

4. **Compile the production package**:
   ```bash
   npm run build
   ```

---

## 🚀 Deployment

This project is fully optimized and configured for seamless deployment to **Vercel** out-of-the-box. Import the repository directly onto the Vercel Dashboard, and it will configure Vite settings, run the compilation build, and deploy to production within seconds.
